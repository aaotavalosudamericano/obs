<h3>Tipos de datos en Javascript</h3>

1. Número (Number):

```javascript

let num = 42;
```


1. Cadena de texto (String):

```javascript

let string = "Hola, mundo!";
```


1. Booleano (Boolean):

```javascript

let isstudent = true;
let isstudent = false;
```


1. Valor nulo (Null):

```javascript

let nul = null;
```


1. Valor indefinido (Undefined):

```javascript

let undefined;
```


1. Símbolo (Symbol) - Introducido en ECMAScript 6:

```javascript

let symbol = Symbol('descripcion');
```


1. Objeto (Object):

```javascript

let object = {
  name: 'Juan',
  age: 30
};
```


1. Tipo de dato función (Function):

```javascript

let fun = function() {
  console.log('Esto es una función.');
};
```
